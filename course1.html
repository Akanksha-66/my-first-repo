Introduction to Web Development

This webpage is created as part of the Experiment-1 to understand the use of HTML structures like headings, paragraphs, lists, and text formatting tags.

Special characters example: © 2025 MyCollege | HTML uses < and > to display angle brackets.

Objectives

  1. To design a simple course curriculum webpage using HTML.
  2. To practice the use of basic text formatting tags like bold, italic, and underline.
  3. To learn the implementation of ordered, unordered, description, nested, and mixed lists.
  4. To understand webpage content structuring using headings and paragraphs.

Tools Required

  • Computer / Laptop with any OS
  • Text Editor (VS Code, Sublime Text, Notepad++)
  • Web Browser (Chrome, Edge, Firefox)
